# Définition
## Une liste chaînée est une structure de donnée linéaire dans laquelle il y a des noeuds. Chaque noeud contient une donnée et un pointeur vers le noeud suivant. Le dernier élément n'a pas de pointeur. 

# Opérations principales
- insertion: méthode pour insérer un noeud au début
- afficher: méthode pour afficher la liste chainée (ex: 1 -> 2 -> 3 -> None)


# Avantages et inconvénients
## Avantages
- Insertion et suppression rapide
- Les listes chaînées peuvent s'agrandir et rétrécir indéfiniment

## Inconvénients
- Il faut parcourir toute la liste pour trouver un élément
- Utilise beaucoup de mémoire


# Exemples d'utilisation
- Historique de navigation
- Fonctionnalité annuler/rétablir
